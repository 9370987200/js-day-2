5.Explain prototypes

Every object in javascript has a build in property, which is called its prototype.The prototype is itself an object ,so the prototype will have its own prototype , making what's called prototype chain. The chain ends when we reach a prototype that has null for its own prototype.

function brother(first,last,age){
        this.firstname=first
        this.lastname=last
        this.age=this.age
    }
    brother.prototype.nationality="english"
    const mybrother= new brother("shubham","bhagyawant",25)
    console.log('the nationality of my brother is '+ mybrother.nationality)
    output:the nationality of my brother is english
